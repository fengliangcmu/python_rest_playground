REST API examples

  • http://localhost:8888/
  • http://localhost:8888/home
  • Get query para from url: http://localhost:8888/user/feng1
  • Get query para of a certain type: http://localhost:8888/post/111111
  • POST a Form: http://localhost:8888/submitForm
  • Upload a file to folder uploadedFiles: http://localhost:8888/uploadFile
  • POST a json in body: http://localhost:8888/getJson
  • Show how to connect to mongodb : http://localhost:8888/connectMongo

Service APIs:

All support POST method; Some support DELETE; Route SaveFile supports GET which display a page to upload files.

fileType is : data / model

isUsingCloud is : True / False

  • http://localhost:8888/app/createApp
    #Json payload must include : appName appDesc username
  • http://localhost:8888/app/updateApp

  • http://localhost:8888/app/deleteApp
    #Json payload must include : appName username
  • http://localhost:8888/app/deleteAllAppForUser

  • http://localhost:8888/file/saveFile
